## 💡 Developer Philosophy

> Code is read far more often than it's written — I optimize for the next
> developer, even when that developer is future me.

- **Clarity over cleverness** — simple, maintainable solutions win
- **APIs are contracts** — design them like someone else's product depends on it
- **Measure, don't guess** — performance and scale decisions backed by data
- **Ship, then refine** — momentum beats perfectionism
